AYUDA FORMULARIO PUESTO
En este formulario de puesto el usuario podr realizar lo siguiente:
Insertar: Al presionar este botn se habilitarn todos los campos excepto el campo
Cdigo, en forma vaca para que el usuario pueda ingresar un nuevo registro.
Modificar: Al presionar este botn se habilitarn todos los campos excepto el
campo Cdigo, del registro en el que se encuentre posicionado actualmente, y
podr ser modificado, segn lo quiera el usuario.
Eliminar: Al presionar este botn se mostrar un mensaje de advertencia al
usuario, si el usuario acepta, se eliminar el registro en el cual el usuario se
encuentre posicionado actualmente.
Guardar: Si el usuario desea guardar la insercin de un nuevo registro o la
modificacin de uno ya existente, debe presionar este botn para que los datos
queden registrados en la base de datos.
Cancelar: Al presionar este botn se cancelar toda accin que se est realizando
en el momento.
Refrescar: se actualizar nuevamente el formulario.
Consultar: se le mostrar una nueva ventana donde podr consulares datos
especficos del formulario.
Botones de gua: podr navegar entre los registros de la tabla que se le muestran.
Ayuda: se abrir el archivo de ayuda sobre el formulario.
Salir: se cerrar el formulario.
Campo Cdigo: El sistema generara un cdigo automtico que corresponda al numero del
registro que correspondera.
Campo Puesto: Campo que contendr el nombre del puesto que se desea almacenar o
modificar.
Campo Descripcin: Campo que contendr una breve descripcin sobre el puesto.
Campo Salario: Este campo contendr el salario que este puesto ofrece.
Campo Estado: se debe seleccionar activo para que el campo se almacene en la base de
datos y este pueda ser visible, pueda ser utilizado en registros futuros.
